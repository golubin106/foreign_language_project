# LinguaLearn

Дипломный проект: интерактивная платформа для изучения английского языка.

## Что можно показать на предзащите

- Главная страница с описанием проекта.
- Каталог уроков с поиском, фильтром по уровню и сортировкой.
- Страница урока со списком новых слов.
- Тест по уроку с подсчетом результата.
- Регистрация и вход пользователя.
- Личный кабинет с прогрессом обучения и историей прохождения тестов.
- Админ-панель для добавления, редактирования и удаления уроков.
- Управление ролями пользователей в админ-панели.
- Серверная часть с файловой базой данных для пользователей, уроков и результатов.

## Как запустить

Установите зависимости:

```bash
npm install
```

Запустите сервер и интерфейс одной командой:

```bash
npm run dev:full
```

После запуска откройте адрес, который покажет Vite, обычно это `http://localhost:5173`.

Сервер работает на `http://localhost:3001`. При первом запуске он создаст файл базы данных `server/data/database.json`.

Если нужно запустить части проекта отдельно, используйте две команды в двух окнах терминала:

```bash
npm run dev:api
npm run dev
```

## Доступ администратора

Администратор теперь создается без демонстрационного кода. В пустой базе данных первый зарегистрированный пользователь автоматически получает роль `admin`, а все следующие пользователи получают роль `student`.

После входа администратора в меню появится ссылка "Админ". В админ-панели можно управлять уроками и менять роли других пользователей.

Если вы уже запускали старую версию проекта и хотите начать демонстрацию с чистой базы, остановите проект и удалите файл `server/data/database.json`. После следующего запуска зарегистрируйте первый аккаунт, и он станет администратором.

## Демонстрационный сценарий

1. Запустить проект командой `npm run dev:full`.
2. Открыть главную страницу и кратко объяснить цель проекта.
3. Зарегистрировать первый аккаунт: он станет администратором.
4. Перейти в каталог уроков и показать поиск/фильтр.
5. Открыть любой урок, например `Lesson 1: Greetings`.
6. Пройти тест, чтобы результат сохранился на сервере.
7. Открыть личный кабинет и показать сохраненный прогресс.
8. Перейти в админ-панель, добавить новый урок и показать его в каталоге.
9. Зарегистрировать второго пользователя и через админ-панель показать изменение его роли.

## Что можно доработать к финальной защите

- Заменить файловую базу на SQLite или PostgreSQL.
- Добавить восстановление пароля и подтверждение email.
- Расширить тесты несколькими типами заданий.
- Добавить статистику по темам, уровням и датам.
- Улучшить адаптивность и визуальное оформление.
