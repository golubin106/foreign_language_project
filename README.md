# LinguaLearn

Дипломный проект: интерактивная платформа для изучения английского языка.

## Что можно показать на предзащите

- Главная страница с описанием проекта.
- Каталог уроков с поиском, фильтром по уровню и сортировкой.
- Страница урока со списком новых слов.
- Тест по уроку с подсчетом результата.
- Регистрация и вход пользователя.
- Личный кабинет с прогрессом обучения и историей прохождения тестов.
- Админ-панель для добавления и редактирования уроков.

## Как запустить

```bash
npm install
npm run dev
```

После запуска откройте адрес, который покажет Vite, обычно это `http://localhost:5173`.

## Демонстрационный сценарий

1. Открыть главную страницу и кратко объяснить цель проекта.
2. Перейти в каталог уроков и показать поиск/фильтр.
3. Открыть любой урок, например `Lesson 1: Greetings`.
4. Пройти тест и получить результат.
5. Зарегистрироваться, открыть личный кабинет и показать сохраненный прогресс.
6. Перейти в админ-панель, добавить новый урок и показать его в каталоге.

## Что можно доработать к финальной защите

- Подключить серверную часть и базу данных.
- Добавить роли пользователя и администратора.
- Расширить тесты несколькими типами заданий.
- Добавить статистику по темам, уровням и датам.
- Улучшить адаптивность и визуальное оформление.
